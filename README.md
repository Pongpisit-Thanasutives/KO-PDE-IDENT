# KO-PDE-IDENT

This repository provides the Python code implementing the KO-PDE-IDENT framework.

## Reproducing the Results

Run the following notebooks to reproduce the results presented in the paper:

1. `FS/Burgers Screening using Knockoffs-econ.ipynb`
2. `FS/KS Screening using Knockoffs-econ.ipynb`
3. `FS/RD Screening using Knockoffs-econ.ipynb`
4. `FS/GS Screening using Knockoffs-econ.ipynb`
5. `FS/KdV Screening using Knockoffs-econ.ipynb`
